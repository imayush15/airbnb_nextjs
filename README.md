## Run Locally

Clone the project

```bash
  git clone https://github.com/imayush15/travelsome-nextjs.git
```

Go to the project directory

```bash
  cd travelsome-nextjs
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
